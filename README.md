## 影视CMS资源

### 外部接口

**接口参数**

```
ac    list/videolist 视频列表/详情列表
t     分类id
pg    页数
wd    搜索关键字
h     几小时内的数据
ids   多个id逗号分割
```

**例子**

```
?ac=list&t=1&pg=5       分类1的数据第5页
?ac=list&h=24           24小时内的视频列表
?ac=videolist&ids=6,7   视频6和7数据
?ac=videolist&h=24      24小时内的视频数据
```
